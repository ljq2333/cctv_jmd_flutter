# CCTV节目单查询软件 - 需求文档

## 1. 项目概述

### 1.1 项目名称
CCTV节目单查询软件（cctv_jmd_flutter）

### 1.2 项目描述
基于Flutter开发的跨平台CCTV节目单查询应用，支持查看CCTV全频道节目单、搜索节目、收藏节目、节目提醒等功能。

### 1.3 目标平台
- Android
- Windows
- Web

---

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 节目单查看
- 支持查看CCTV1-CCTV17所有频道节目单
- 显示节目名称、播出时间、时长
- 支持查看历史节目单（前一个月）和未来节目单（后两天）
- 高亮显示当前正在播出的节目

#### 2.1.2 频道切换
- 侧边栏显示频道列表
- 支持滑动打开侧边栏切换频道
- 记住用户最后选择的频道

#### 2.1.3 节目搜索
- 支持按节目名称搜索
- 搜索范围可选：当前频道 / 所有频道
- 默认搜索范围为当前频道
- 显示搜索结果的频道、时间、节目名称

#### 2.1.4 节目收藏
- 支持收藏喜欢的节目
- 收藏列表管理（查看、删除）
- 收藏数据本地持久化存储

#### 2.1.5 节目提醒
- 支持为指定节目设置提醒
- 提醒方式：
  - 系统推送通知（Android/Windows）
  - 应用内提醒
  - 写入系统日历（Android/Windows）
- 提醒时间可设置（默认节目开始前5分钟）

#### 2.1.6 直播入口
- 提供CCTV各频道直播观看入口
- 跳转到CCTV官方直播页面

### 2.2 辅助功能

#### 2.2.1 离线支持
- 缓存已查询的节目单数据
- 无网络时可查看缓存的节目单
- 自动同步最新数据（联网时）

#### 2.2.2 主题设置
- 支持浅色模式
- 支持深色模式
- 支持跟随系统设置
- 用户可手动切换

## 3. 技术架构

### 3.1 技术栈
- **框架**: Flutter 3.x
- **语言**: Dart
- **状态管理**: 待定（Provider/Riverpod/Bloc）
- **本地存储**: SharedPreferences / Hive / SQLite
- **网络请求**: Dio / http
- **日历集成**: device_calendar（Android/Windows）

### 3.2 API接口

#### 节目单查询接口
```
GET https://api.cntv.cn/epg/getEpgInfoByChannelNew
```

**参数说明**:
| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| c | string | 频道ID | cctv1, cctv2, ... |
| serviceId | string | 服务ID | tvcctv |
| d | string | 日期 | 20260607 |
| t | string | 返回格式 | jsonp |
| cb | string | 回调函数名 | setItem1 |

**响应数据结构**:
```json
{
  "data": {
    "cctv1": {
      "channelName": "CCTV-1 综合",
      "isLive": "当前直播节目名",
      "liveSt": 1780830000,
      "lvUrl": "https://tv.cctv.com/live/cctv1",
      "list": [
        {
          "title": "节目名称",
          "startTime": 1780763470,
          "endTime": 1780766170,
          "showTime": "00:31",
          "length": 2700,
          "column_url": "栏目URL",
          "columnBackvideourl": "视频URL"
        }
      ]
    }
  }
}
```

## 5. 数据存储

### 5.1 本地存储结构

#### 收藏数据
```json
{
  "favorites": [
    {
      "id": "unique_id",
      "channelId": "cctv1",
      "channelName": "CCTV-1 综合",
      "title": "早间新闻",
      "startTime": 1780763470,
      "date": "20260607"
    }
  ]
}
```

#### 提醒数据
```json
{
  "reminders": [
    {
      "id": "unique_id",
      "channelId": "cctv1",
      "title": "早间新闻",
      "startTime": 1780763470,
      "reminderTime": 1780763170,
      "enabled": true
    }
  ]
}
```

#### 缓存数据
```json
{
  "cache": {
    "cctv1_20260607": {
      "data": {...},
      "timestamp": 1780830000
    }
  }
}
```

#### 用户设置
```json
{
  "settings": {
    "theme": "system",
    "lastChannel": "cctv1",
    "reminderEnabled": true,
    "reminderMethods": ["notification", "inApp", "calendar"],
    "reminderMinutesBefore": 5
  }
}
```

## 6. 非功能需求

### 6.1 性能要求
- 首页加载时间 < 2秒
- 频道切换响应时间 < 500ms
- 搜索响应时间 < 1秒

### 6.2 兼容性要求
- Android: API 21+ (Android 5.0+)
- Windows: Windows 10+
- Web: 现代浏览器（Chrome、Firefox、Edge、Safari）

### 6.3 离线要求
- 缓存最近7天的节目单数据
- 无网络时显示缓存数据并提示用户
- 联网后自动同步最新数据
